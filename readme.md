# Form-Builder
### A package to simplify the way that you create your HTML forms!

## Installation

[![NPM Version][npm-image]][npm-url]

In a browser:
```html
<script src="formBuilder.js"></script>
```

Using npm:
```shell
$ npm install --save htmlform-builder
```
